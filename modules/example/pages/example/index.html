Example Module

This page is served by the example frontend module at modules/example/pages/example/index.html. If you can see it, the module wiring works: page discovery walks modules/example/pages/ in addition to the app's own content/pages/, and module.js was bundled via import.meta.glob.

Open the browser console — the module's module.js logs a line at startup so you can confirm the JS side wired up too.
